# Glossary

## App Router
Next.js routing system introduced to simplify nested layouts and server components.

## Feature-based architecture
A structure where application features own their own components, hooks, and logic.

## Route group
A Next.js App Router construct such as `(auth)` or `(dashboard)` that groups routes without affecting the URL.

## ADR
Architecture Decision Record: a document that records important architecture and design decisions.

## Shared utility
Code in `lib/`, `utils/`, or `hooks/` intended to be reused across multiple features.
