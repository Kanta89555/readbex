# Overview

## Purpose

This repository includes a starter Next.js application structure designed for scalability and clarity. The `my-app` folder demonstrates a feature-based architecture with a clean App Router setup.

## Scope

- Modern React with Next.js App Router
- Feature-driven organization
- Shared UI components and layout primitives
- Basic placeholders for authentication, dashboard, and utility logic

## Current contents

- `my-app/app/` contains the main application route tree
- `my-app/features/` contains `auth`, `dashboard`, and `products` modules
- `my-app/lib/` contains shared services and constants
- `my-app/components/` contains reusable UI components

## Notes

- This repository is currently a starter scaffold and is intended to be extended with real domain features, styling, and backend integration.
- Document architecture decisions in `docs/adr/` as the design evolves.
